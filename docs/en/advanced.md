# Tab: Advanced

The **Advanced** tab contains special settings for diagnostics and debugging.

**Features:**

- **Verbose Logging:** Enable this option to get especially detailed log output. This is useful if you want to analyze problems or report errors to the developer.

**Note:**
Advanced logs should only be enabled when needed, as they can produce a lot of additional information in the ioBroker log.
